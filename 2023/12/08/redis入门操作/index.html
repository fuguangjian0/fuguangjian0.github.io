redis入门操作

Redis入门操作

基于内存的key-value结构数据库, 用途广泛的存储中间件

NoSQL, 非关系型数据库, 是对关系型数据库的补充

数据类型

Redis是 k-v 结构数据, key是String, value有5种数据类型

字符串 String
哈希 hash 又名散列, 类似HashMap
列表 list 按照插入顺序排序, 可以重复元素, 类似LinkedList
集合 set 无序集合, 没有重复, 类似HashSet
有序集合 sorted set/zset 集合每个元素关联一个分数, 根据分数升序排序, 没用重复

命令

字符串

GET key 获取a的值
SET key value 设置a的值
SETEX key seconds value 设置a的值, 设置过期时间(s)

hash

HGET key field 获取key的field
HSET key field value 设置key的field 和 value
HDEL key field 删除key的field
HKEYS key 获取key的所有field
HVALS key 获取key的所有value

list

LPUSH key value1 [value2] 插入k-v
LRANGE key start stop 获取指定下标范围的value
RPOP key 移除并获取列表最后一个value
LLEN key 获取列表长度
BRPOP key1 [key2 ] timeout 移出并获取列表的最后一个元素， 如果列表没有元素会阻塞列表直到等待超 时或发现可弹出元素为止

set

SADD key member1 [member2] 向集合添加一个或多个成员
SMEMBERS key 返回集合中的所有成员
SCARD key 获取集合的成员数
SINTER key1 [key2] 返回给定所有集合的交集
SUNION key1 [key2] 返回所有给定集合的并集
SREM key member1 [member2] 移除集合中一个或多个成员

zset

ZADD key score1 member1 [score2 member2] 向有序集合添加一个或多个成员
ZRANGE key start stop [WITHSCORES] 通过索引区间返回有序集合中指定区间内的成员
ZINCRBY key increment member 有序集合中对指定成员的分数加上增量 increment
ZREM key member [member …] 移除有序集合中的一个或多个成员

通用

KEYS pattern 查找所有符合给定模式( pattern)的 key
EXISTS key 检查给定 key 是否存在
TYPE key 返回 key 所储存的值的类型
DEL key 删除 key
